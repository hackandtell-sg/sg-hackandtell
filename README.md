# Hack && Tell Singapore

This is the source for the [website of the Singapore chapter][htsg] of [Hack && Tell][ht].

## Run locally

1. Install [Jekyll][]

        gem install jekyll
        
   **or** use Nix
   
        nix-shell

2. Run the jekyll server

        jekyll serve

[ht]: https://hackandtell.org/
[htsg]: https://sg.hackandtell.org/
[Jekyll]: https://jekyllrb.com
